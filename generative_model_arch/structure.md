# Technical Documentation: Minimax KSI Project Files

This document provides a strict, technical explanation of the specific role each file plays in executing the Minimax-Optimal Kernelized Stochastic Interpolant (KSI) framework.

## 1. Configuration & Data Management

### `configs/default_config.yaml`

This file stores the global hyperparameters and mathematical constraints required for the algorithm. It explicitly defines the ambient spatial dimension ($p$), the estimated intrinsic manifold dimension ($d$), the target smoothness regularity ($\beta$), the number of local charts ($m$), the truncation limit for the wavelet expansion ($P$), and the number of discrete integration steps for the stochastic differential equation (SDE).

### `data/raw/` and `data/processed/`

These directories manage the state of the empirical data. `raw/` stores the target point cloud $X \in \mathbb{R}^p$. `processed/` stores the exact same data after it has been partitioned into $m$ distinct subsets, ensuring each subset satisfies the local injectivity radius $\tau$ required for valid manifold reconstruction.

## 2. Core Source Modules (`src/`)

### `src/features/wavelet_map.py`

This file replaces the neural network generator. It implements the deterministic feature map $\phi(x): \mathbb{R}^p \to \mathbb{R}^P$. Using a truncated Besov wavelet basis or a Wavelet Scattering Transform, it projects ambient data into a finite-dimensional feature space. This file strictly enforces the condition that the resulting hypothesis space matches the intrinsic regularity $\mathcal{H}_1^{d/2}$.

### `src/solver/clustering.py`

This module contains the spatial partitioning algorithms (such as K-Means or manifold-aware clustering). Its sole operational purpose is to divide the ambient target data into localized Euclidean patches, preventing the global regression system from smoothing over complex topological structures (holes, disconnected components).

### `src/solver/local_regression.py`

This file replaces the WGAN adversarial min-max optimization. For a given local chart $i$ and time step $t$, it constructs the empirical $P \times P$ Gram matrix $\hat{K}_t^{(i)}$ from the feature gradients $\nabla\phi_i(X)$ and solves the linear system $\hat{K}_t^{(i)} \eta_t^{(i)} = \hat{r}_t^{(i)}$ using `torch.linalg.solve`. It outputs the optimal coefficients $\eta_t^{(i)}$ for the local drift field.

### `src/dynamics/vector_field.py`

This module handles topological gluing in the velocity domain, replacing Fefferman's computationally heavy spatial reconstruction maps. It implements the smooth partition of unity $\rho_i(x)$ to calculate the global drift $\hat{b}_t(x)$ by executing a convex combination of the local vector fields $\hat{b}_t^{(i)}(x)$. This ensures the resulting flow is globally $(\beta+1)$-smooth.

### `src/sampler/diffusion_sched.py`

This file computes the theoretical optimal diffusion coefficient $D_t^*$ at any given time step $t$. It calculates $D_t^* = \frac{\alpha_t \gamma_t}{\beta_t}$ strictly following the interpolant variance schedule. This coefficient determines the exact magnitude of stochastic noise required to bound the estimation errors generated by the linear projection.

### `src/sampler/sde_integrator.py`

This module executes the generative process. It implements the Euler-Maruyama numerical integration scheme to simulate the SDE: $dX_t = \hat{b}_t(X_t)dt + \sqrt{2 D_t^*} dW_t$. It transports samples from the standard normal prior $\mathcal{N}(0, I_p)$ onto the target manifold by iterating over the precomputed velocity fields and injecting the noise scheduled by `diffusion_sched.py`.

## 3. Execution Scripts (`scripts/`)

### `scripts/01_cluster_data.py`

The initial data preparation script. It imports the raw data, executes the clustering algorithm from `src/solver/clustering.py`, and saves the partitioned geometric subsets into the `data/processed/` directory.

### `scripts/02_solve_velocities.py`

The linear regression solver script. It iterates over a discretized time grid $t \in [0,1]$. At each step, it calls `local_regression.py` for all $m$ charts, generating and saving the sequence of coefficient matrices $[\eta_{t_0}^{(i)}, \dots, \eta_{t_N}^{(i)}]$. This script functionally replaces the "training" phase of a standard generative model.

### `scripts/03_generate_samples.py`

The inference script. It initializes an array of random noise vectors $Z \sim \mathcal{N}(0, I_p)$, loads the precomputed coefficients $\eta_t$, and utilizes `sde_integrator.py` to push the noise through the continuous-time dynamics. The output is the generated synthetic data strictly supported on the unknown manifold.
